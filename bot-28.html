BotCity Instagram ChatGpt
by Thiago Noronha on 01/16/2023
Initial Setup:
You need to create a BotCity account and download the project resources: https://lnkd.in/dAae4ZgS
You must have an OpenAI key. You can get one here: https://lnkd.in/dQJb7GaZ
You must have a Google API key and password: https://lnkd.in/dxKKKa4R
You must also have an Instagram Account.

How does it work:
After you provide the login and password, the software will store the data in a folder called Chave;
You can select a subject making sure to provide the maximum detail you can about what you would like to talk;
You can ask for hashtags to be added at the end of the text;
You can describe the image that you would like to use for your post and Google Images will return an image related to the subject;

(It is not yet possible to visualize the image before posting. It will be added in future versions).
The software will open Instagram, select the Google Images file and write the text generated by OpenAI.
